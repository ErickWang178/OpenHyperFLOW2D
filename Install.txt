System Requirements
===========================

* CPU:           x86_64 compatible;
* GPU:           CUDA compatible NVIDIA card;
* OS:            Linux x86_64 (tested on OpenSUSE 10+, SLES 10+, CentOS 6+, Scientific Linux 5+);
* C++ compiler:  GNU C++ 4.1+, Intel C++ 10+;
* Drivers        Latest proprietary Nvidia driver
* SDK            CUDA 5.5+  (up to CUDA 8.0 tested)

Installation
============

1. Configure C++ compiler

Edit '.compiler' file:

Select compiler:
 'COMPILER=GNU' for GNU C++ compiler or
 'COMPILER=INTEL' for Intel C++ compiler

2. Configure parallel features

Edit '.parallel' file:

Select 'PARALLEL=CUDA'

Edit 'cuda.compiler' file:
Specify CUDA version: e.g. 'CUDA_VER=8.0' 
Specify CUDA root path: Default 'CUDA_ROOT=/usr/local'

3. Build

Run command 'make build' of 'make rebuild' for build OpenHyperFLOW2D

4. Run Test Cases

Run command 'make test' for run all test cases or run

  'make ObliqueShock' for Oblique Shock test case

  'make Wedge' for Wedge test case

  'make Step' for Mach 3 forward step test case
